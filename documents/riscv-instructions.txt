
RISC-V instructions: 
ADDI, SLTI, SLTIU, ANDI, ORI, XORI, SLLI, SRLI, SRAI, LOAD, LUI, AUIPC,
ADD, SLT, SLTU, AND, OR, XOR, SLL, SRL, SRA, SUB, JALR, STORE,
JAL, BEQ, BNE, BLT, BGE, BLTU, BGEU

* The position and width of all instruction fields (funct7, funct3, rs2/shamt, rs1, rd, opcode, imm) 
are depicted in page 120, table at the top of the page. 
Funct12 used for SYSTEM instructions ECALL and EBREAK is the same field as the imm[11:0] in the I-type.

* All instructions share the same 7-bit opcode field in the lower bits.
	- Lower 2 bits of opcode are 11b (binary) for all RV32I instructions.
	- The higher 5 bits of opcode determine the major opcodes, see table 24.1
	opcode LOAD = 		0000011
	opcode MISC-MEM = 	0001111
	opcode OP-IMM = 	0010011
	opcode AUIPC = 		0010111
	opcode STORE =		0100011
	opcode OP = 		0110011
	opcode LUI = 		0110111
	opcode BRANCH =		1100011
	opcode JALR = 		1100111
	opcode JAL = 		1101111
	opcode SYSTEM = 	1110011
	The other opcodes are not part of RV32I
